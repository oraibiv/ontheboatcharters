Setup Instructions

Email uses Gmail SMTP with an App Password (not your regular Gmail password). This keeps things secure — the app password only works for sending email.

Step 1: Create a Gmail App Password

Go to myaccount.google.com/apppasswords (you need 2-Step Verification enabled on the Gmail account). Click "Select app" → choose "Other" → type "On The Boat" → click Generate. You'll get a 16-character password like abcd efgh ijkl mnop.

Step 2: Set Environment Variables

Before starting the app, set these in your terminal: 

export GMAIL_ADDRESS=yourname@gmail.com
export GMAIL_APP_PASSWORD=abcdefghijklmnop
export CAPTAIN_EMAIL=captain@ontheboatcharters.com

python3 app.py

Tip: Put the export lines in a file called .env.sh and run source .env.sh before starting the app.

Step 3: Set Up Daily Reminders (cron)

The booking confirmation and cancellation emails send automatically. For the 24-hour reminders and captain morning digest, set up a daily cron job: 

# Edit crontab
crontab -e

# Add this line (runs at 6 AM daily):
0 6 * * * cd /path/to/ontheboat && source .env.sh && /path/to/venv/bin/python send_reminders.py
